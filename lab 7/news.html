20.09.2018

Відбулися відкриті змагання "OPEN CUP" Run Base Kyiv running club

У манежі вирували неабиякі емоції: спортсмени серйозно боролися за перемогу, навіть застосовували спурт на фініші, а вболівальники палко підтримували спортсменів! Нагородження проводилося серед чоловіків і жінок у двох категоріях - до 35 років і після 35 років. Переможці отримали медалі, дипломи та невеличкі приємні подарунки від клуба. Ну а після змагань усі дружно закривали «вуглеводне вікно» Дякуємо нашим тренерам Ірині Ліщинській та Олександрі Азаровій та усім учасникам змагань за супердрайвову атмосферу та позитив! Усім біг!

30.09.2018

Run Base Kyiv провів змагання серед членів клубу

25 січня відбувся відкритий кубок Run Base Kyiv. У легкоатлетичному манежі “Атлет” любителі змагалися на дистанціях 2 та 5 км. Змагання дали шанс членам клубу оцінити власні сили та можливості, підвести певний підсумок досягнень за пройдений період тренувань. Найбільш спритні учасники (1, 2 і 3 місця) отримали грамоти з відзнаками, а перші місця були нагороджені статуетками із зображенням логотипу клубу Run Base Kyiv. Окрім того, переможці отримали 50% знижку на функціональне обстеження.
